##<a id="1">1.获取游客ID</a>

###<a id="1.1">URL</a>
visitor/getVid.json

###<a id="1.2">请求方式</a>
GET

###<a id="1.3">Header</a>
Content-Type : application/json

###<a id="1.4">请求参数</a>
无

###<a id="1.5">返回Json示例</a>
####<a id="1.5.1">请求成功</a>
	{
		"success":"true",
		"data":{
		  "vid":33234234   //游客id
		}
	}

####<a id="1.5.2">请求失败</a>
	{
		"error_code":"10000",
		"error_message":"XXXXX"
	}
[错误码详见错误码对照表](错误码对照表.md)

Content-Type : application/json
