###<a id="1.4">请求参数</a>
     参数      | 必选 			| 类型及范围     |说明
------------- | ------------- | -------------|---------- 
source		    | true		   | string       |来源，如安卓，iOS
userId	       | true		   | string       |uid
token			|true         |string       |校验token

###<a id="1.6">返回字段说明</a>
     参数      | 类型及范围     |说明
------------- | -------------|---------- 
success		 | string       |请求成功
userId		 	 | long       |uid
userName		 | string       |用户名
remarkName	 | string       |备注名
portraitSmall | string       |头像小图url
firstCharacter| string       |用户名或备注名首写字母(大写),如张三则为ZS,Peter则为P
