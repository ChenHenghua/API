##<a id="1">加好友请求</a>

###<a id="1.1">URL</a>
[friend/add_friend.json](http://api.maitian.com/v1/friend/add_friend.json)

###<a id="1.2">请求方式</a>
POST

###<a id="1.3">Header</a>
Content-Type : application/json

###<a id="1.4">请求参数</a>
     参数      | 必选 	    | 类型及范围     |说明
-------------  | ---------- | -------------  |---------- 
userId         | true	      | long         |用户id
token          | true	      | string         |用户token
toId          | true         | long         |被加好友的id
text          | true         | string       |添加好友时发送的信息
remark        | true         | string       |好友在通讯录中的名字,作为备注名

###<a id="1.5">请求Json示例</a>
	{
		"userId" : 23423322,
		"token" : "3dsfsdfsdfsd534534",
		"toId" : 1552720789,
		"text" : "我是岩途",
		"remark" : "傻二狗"
	}

###<a id="1.6">返回Json示例</a>
####<a id="1.6.1">请求成功</a>
	{
		"success":"true"
	}

####<a id="1.6.2">请求失败</a>
	{
		"error_code":"10000",
		"error_message":"XXXXX"
	}
[错误码详见错误码对照表](错误码对照表.md)
