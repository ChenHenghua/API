##<a id="1">修改昵称</a>

###<a id=1.1">URL</a>
user/settings/nickname.json

###<a id="1.2">请求方式</a>
POST

###<a id="1.3">请求参数</a>
     参数    | 必选 			| 类型及范围    | 说明
------------ | -----------| ------------- |---------- 
source		| true		| string        | 来源，如安卓，iOS
userId  	| true		| long          | 用户ID
token		| true		| string        | 用户token
nickName	| true		| string	| 新的昵称

###<a id="1.4">请求Json示例</a>
	{
		"source":"Android",
		"userId":111111,
		"token":"asdfasexx",
		"nickName":"hltc"
	}

###<a id="1.5">返回Json示例</a>
####<a id="1.5.1">请求成功</a>
	{
		"success":"true"
	}

####<a id="1.5.2">请求失败</a>
	{
		"error_code":"?",
		"error_message":"?"
	}
[错误码详见错误码对照表](错误码对照表.md)

###<a id="1.1.6">返回字段说明</a>
     参数     	| 类型及范围     |说明
------------- 	| -------------|---------- 
success		| string       |请求成功
